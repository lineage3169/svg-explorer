# SVG Explorer | 圖示搜尋工具

SVG Explorer 是一個輕量且現代化的本地端 SVG 圖示預覽與搜尋工具。只需透過瀏覽器打開，即可輕鬆載入本地資料夾中的所有 SVG 檔案，讓開發者與設計師能更有效率地管理與尋找專案中的圖示。

## ✨ 功能特色

- **📁 批次載入**：支援選取本地資料夾（包含子目錄），批次讀取並顯示所有 SVG 圖示。
- **🔍 即時搜尋**：提供搜尋列，可依據圖示檔名進行即時過濾。
- **👁️ 直覺預覽**：以舒適的網格版面呈現圖示預覽，支援深色模式介面設計。
- **📋 一鍵複製**：點擊任意圖示即可快速複製其檔案名稱（不含 `.svg` 附檔名），方便開發時直接貼上使用。
- **⚡ 極致輕量**：採用單一 HTML 檔案實作，無須建置(Build)過程或安裝複雜的 NPM 依賴，隨開即用。
- **🛡️ 效能優化**：針對大量檔案採用分批讀取（Chunking）機制，避免載入時造成畫面卡頓。

## 🚀 快速開始

1. 將本專案下載到您的電腦中。
2. 使用任何現代主流瀏覽器（Chrome, Edge, Firefox, Safari 等）直接開啟 `index.html` 檔案。
3. 點擊畫面中的 **「選擇資料夾」** 按鈕，選取您存放圖示的目錄。
4. 瀏覽器可能會跳出讀取權限確認，請允許讀取。
5. 成功載入後即可開始預覽或搜尋圖示。

## 🛠️ 技術選型

- **核心架構**：HTML5, CSS3, JavaScript (ES6+)
- **前端框架**：Vue 3 (透過 CDN 引入，負責狀態管理與資料綁定)
- **字體設計**：Google Fonts (Inter)
- **Web APIs**：支援 `webkitdirectory` 的 File API, FileReader API, Clipboard API

## 💡 注意事項

- 本工具完全在前端（您的瀏覽器內）運行，所選取的資料夾與圖示內容**不會**被上傳至任何伺服器，請安心使用。
- 部份瀏覽器在開啟本地 `file://` 協定時，可能會對檔案讀取有較嚴格的安全限制。若遇到無法讀取的情況，建議可以使用像是 VS Code 的 Live Server 擴充功能來運行。
